# PWM Converter

Конвертер диапазона PWM-сигнала на базе **ESP32-C3 Super Mini**.

Преобразует стандартный RC-диапазон 1000–2000 мкс в расширенный диапазон 500–2500 мкс (подходит для цифровых сервоприводов с широким диапазоном). Опционально отображает входное и выходное значения на OLED-дисплее SSD1306.

![Scheme IRL](docs/img/converter.jpg)

## Характеристики

| Параметр | Значение |
|---|---|
| МК | ESP32-C3 Super Mini |
| Вход PWM | 1000–2000 мкс |
| Выход PWM | 500–2500 мкс |
| Частота выходного PWM | 50 Гц |
| Разрешение LEDC | 14 бит |
| Дисплей | SSD1306 128×64 (I2C, 0x3C) *(опционально)* |
| Частота обновления дисплея | ~20 Гц |

## Пины

| Функция | GPIO |
|---|---|
| PWM вход | GPIO 2 |
| PWM выход | GPIO 3 |
| I2C SDA (дисплей) | GPIO 7 |
| I2C SCL (дисплей) | GPIO 6 |

> Пины I2C задействованы только при сборке с флагом `USE_DISPLAY`.

## Принцип работы

1. Входной PWM-сигнал измеряется аппаратным прерыванием (`CHANGE`) — фиксируется время переднего и заднего фронта.
2. Ширина импульса линейно пересчитывается из диапазона IN → OUT.
3. Выходной сигнал формируется через ESP-IDF LEDC (14-битный таймер, 50 Гц).
4. *(При включённом дисплее)* SSD1306 обновляется ~20 раз в секунду: IN/OUT значения в мкс и прогресс-бар выхода.

## Конфигурация

Все параметры хранятся в `include/config.h`.

### Диапазоны PWM

```c
// Входной диапазон (мкс)
#define IN_MIN   1000
#define IN_MAX   2000

// Выходной диапазон (мкс)
#define OUT_MIN  500
#define OUT_MAX  2500
```

Примеры распространённых конфигураций:

| Применение | IN_MIN | IN_MAX | OUT_MIN | OUT_MAX |
|---|---|---|---|---|
| Стандарт RC → стандарт RC | 1000 | 2000 | 1000 | 2000 |
| Стандарт RC → расширенный (по умолчанию) | 1000 | 2000 | 500 | 2500 |
| Стандарт RC → широкий цифровой | 1000 | 2000 | 400 | 2600 |

> `SANITY_MIN` / `SANITY_MAX` вычисляются автоматически как `OUT_MIN − 100` / `OUT_MAX + 100` и защищают от выброса входного сигнала — менять не нужно.

### Пины

```c
#define PIN_PWM_IN   2   // вход PWM
#define PIN_PWM_OUT  3   // выход PWM

#ifdef USE_DISPLAY
#define PIN_SDA  7       // I2C SDA
#define PIN_SCL  6       // I2C SCL
#endif
```

## Флаги сборки

| Флаг | По умолчанию | Описание |
|---|---|---|
| `USE_DISPLAY` | включён | Компилирует поддержку OLED SSD1306. Убери флаг для сборки без дисплея. |

Управляется в `platformio.ini`:

```ini
build_flags =
    -DUSE_DISPLAY   ; убрать эту строку для сборки без дисплея
```

## Версионирование

Версия прошивки задаётся в `include/version.h` и выводится в Serial при старте:

```
FW v1.1.1 [display: ON]
Ready
```

## Зависимости

- [Adafruit SSD1306](https://github.com/adafruit/Adafruit_SSD1306) `^2.5.7` *(только с `USE_DISPLAY`)*
- [Adafruit GFX Library](https://github.com/adafruit/Adafruit-GFX-Library) `^1.11.9` *(только с `USE_DISPLAY`)*

## Сборка и прошивка

Проект использует [PlatformIO](https://platformio.org/).

```bash
pio run --target upload
pio device monitor
```

Скорость монитора: **115200** бод.
